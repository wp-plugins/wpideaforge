Let's user suggest Ideas for upcoming Articles. 

== Description ==

This Plugin adds a Form to an existing Page. All you have
to do is to add the Shortcode [wpideaforge] to that Page.
After that your Visitors can suggest Ideas for Articles
you could write in the near Future. Other Visitors can
vote for Ideas so you can see what is popular, and what is not.

== Installation ==

1. Unzip the ZIP plugin file
2. Copy the `wpideaforge` folder into your `wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add a Page and put [wpideaforge] into the Content
